# House Price Prediction using Machine Learning

This project uses a Linear Regression model to predict house prices based on square footage, number of bedrooms, and bathrooms. It is a beginner-friendly Machine Learning project developed using Python and Scikit-learn in Google Colab.

## Features
- House price prediction
- Linear Regression implementation
- User input prediction system
- Simple and easy-to-understand workflow

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Outcome
Successfully trained a Machine Learning model to predict house prices with user-defined inputs.
